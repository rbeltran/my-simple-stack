# my-simple-stack

A stack in Clojure

## Installation

git clone git@github.com:rbeltran/my-simple-stack.git

Install Lein a tool for compiling and running Clojure projects

http://leiningen.org/

## Usage

compile:

  $ lein compile

run:

  $ lein run
